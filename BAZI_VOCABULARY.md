# 八字解读词汇表

从 `bazi_js` 仓库的 `doc/` 文件夹（37个docx）整理出的所有解读词汇。

---

## 【一、强度类】

| 术语 | 判断规则 |
|------|----------|
| 过旺 | `isWang === 1 && isShouZhi === 0` |
| 过弱 | `isWang === 0 && isShouZhi === 1` |
| 过多 | `occurrences.length >= 3` |
| 重重 | `occurrences.length === 2` |
| 旺相 | `isWang === 1` |

---

## 【二、状态类】

| 术语 | 判断规则 |
|------|----------|
| 透干 | `exists[0] === 1 && exists[1] === 1`（天干有 + 地支有） |
| 虚浮 | `exists[0] === 1 && exists[1] === 0`（只有天干有） |
| 藏干 | `exists[0] === 0 && exists[1] === 1`（只有地支有） |
| 入库 | 天干进入自己的墓库 |
| 出墓 | 冲 + 透干 + 旺相 |
| 空亡 | 年/月/时支：检查自己的地支名是否在**日支的kongWang数组**里；日支：检查自己的地支名是否在**年/月/时支的kongWang数组并集**里 |
| 通根 | 天干与某地支五行相同（如甲寅、乙卯） |

---

## 【三、格局类】

| 术语 | 判断规则 |
|------|----------|
| 开库 | 旺衰适宜 + 透干引动 + 刑冲合会（三要素缺一不可） |
| 入墓 | 能量封闭状态 |

### 墓 vs 库 区别
- **库**：五行旺相时为库，主储藏、积蓄、资源可用（如仓库有物可提取）
- **墓**：五行衰弱时为墓，主封闭、束缚、能量停滞

判断标准：
1. 日干旺相 → 见墓库多为库
2. 日干衰弱 → 见墓库多为墓
3. 藏干透干引动 → 多为库
4. 有刑冲合会引动 → 多为库

## 【四、十神组合类】

### 透干规则（适用于所有组合判断）
- 相邻 + 都透干 = 完整
- 相邻 + 至少一个透干 = 可算
- 仅藏干内部相生（本气→中气→余气）= 不算

| 术语 | 判断规则 |
|------|----------|
| 官杀混杂 | 七杀.exists[0] === 1 && 正官.exists[0] === 1（天干同时有七杀和正官） |
| 并见 | `occurrences.length === 2` |
| 重叠 | `occurrences.length === 2` |
| 过多 | `occurrences.length >= 3` |
| 食神伤官 | 食神存在 && 伤官存在 |
| 正财偏财 | 正财存在 && 偏财存在 |
| 比肩劫财 | 比肩存在 && 劫财存在 |
| 官印相生 | 官星 + 印星相邻（透干规则同上） |
| 食伤生财 | 食伤 + 财星相邻（透干规则同上） |

## 【五、特殊格局】

| 术语 | 判断规则 |
|------|----------|
| 水火既济 | 丙火透干 && 壬水透干 |
| 木火通明 | 甲木透干 && 丙火透干 |
| 金白水清 | 辛金日主 && 壬水透干 && 地支有金根 |
| 专旺 | 某五行特别旺，八字几乎都是生扶它的 |
| 身旺食伤泄秀 | 身强 && 比肩/劫财 && 食神/伤官相邻 |
| 身弱官杀 | 身弱 && 官星与日主相邻 && 无印星与官星相邻 |
| 财多身弱 | 身弱 && 正财/偏财旺 |

## 【六、喜忌神类】

| 术语 | 判断规则 |
|------|----------|
| 官杀为忌 | 官星.xiYong === '忌' |
| 官杀为用 | 官星.xiYong === '用' |
| 印星为忌 | 印星.xiYong === '忌' |
| 印星为用 | 印星.xiYong === '用' |
| 食伤为忌 | 食伤.xiYong === '忌' |
| 食伤为用 | 食伤.xiYong === '用' |
| 旺而无制 | `isWang === 1 && isShouZhi === 0` |
| 弱而受制 | `isWang === 0 && isShouZhi === 1` |
| 透干有根 | `exists[0] === 1 && exists[1] === 1` |
| 透干无根 | `exists[0] === 1 && exists[1] === 0` |
| 旺相透干 | `isWang === 1 && exists[0] === 1 && exists[1] === 1` |

## 【七、神煞类】

神煞类 = 地支.shenSha.includes(神煞名)

包括：桃花、羊刃、建禄、阳刃、文昌、驿马、罗网、亡劫、三刑、六害、自刑

## 附录：现有类属性

### Shishen（十神）属性
| 属性 | 类型 | 说明 |
|------|------|------|
| `name` | string | 十神名称 |
| `exists` | [bool, bool] | [天干存在?, 藏干存在?] |
| `occurrences` | array | 出现位置 [{pillar, occurs, role}] |
| `isWang` | 0/1 | 旺衰（1=旺，0=衰） |
| `isShouZhi` | 0/1 | 受制（1=受制，0=不受制） |
| `xiYong` | string | '用'/'忌'/'闲' |

### Gan/Zhi 属性
| 属性 | 类型 | 说明 |
|------|------|------|
| `name` | string | 名称 |
| `wx` | string | 五行 |
| `yinyang` | string | 阴阳 |
| `pillarIndex` | 0-7 | 数组索引 |
| `hiddenGans` | array | 藏干数组 |
| `kongWang` | array | 空亡地支 |
| `shenSha` | array | 神煞列表 |
