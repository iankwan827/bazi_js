## 速查表（必读）

### 索引对照表

| 名称 | 含义 | 取值范围 |
|------|------|----------|
| **pillars数组索引** | pillars[0-7]，定位具体天干或地支 | 0=年干, 1=年支, 2=月干, 3=月支, 4=日干, 5=日支, 6=时干, 7=时支 |
| **pillar（occurrences）** | occurrences里的pillar值，含义同pillars数组索引 | 0-7 |
| **柱索引（四柱编号）** | 四柱的编号，用于getPillarAt()等 | 0=年柱, 1=月柱, 2=日柱, 3=时柱 |

### 相邻关系（pillars数组索引）

| 类型 | 相邻对 |
|------|--------|
| 同柱相邻 | 0-1, 2-3, 4-5, 6-7 |
| 隔柱相邻(地支与地支) | 1-3, 3-5, 5-7 |

### 十神occurrences结构

```javascript
{
    pillar: 0-7,        // pillars数组索引
    occurs: [1,0],     // [天干存在?, 藏干存在?]
    role: null/'本气'/'中气'/'余气'  // null表示天干
}
```

## 一、整体架构

```
┌─────────────────────────────────────────┐
│           排盘（独立前置步骤）           │
│  输入：出生日期时间 → 输出：命盘JSON     │
└─────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────┐
│          数据层（面向对象）← 当前阶段    │
│  五行 → 天干 → 十神/日元                │
│  地支（冲害破刑）                       │
│  四柱数据组装                           │
└─────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────┐
│          逻辑层（分析计算）              │
│  身强身弱、用神喜忌、大运流年等          │
└─────────────────────────────────────────┘
```

**模块说明：**
- `bazi_classes.js` - 数据层：Element、 Gan、Zhi、Shishen、BaziContext
- `shishen_wangshuai.js` - 十神旺衰计算
- `shishen_pingke.js` - 十神性格特征
- `shishen_geshi.js` - 十神格局分析
- `paipan_node_core.js` - 排盘核心（独立模块）

## 二、数据层架构

### 2.1 类继承关系

```
Element（五行基类）
├── Gan（天干类）
│   └── Shishen（十神/日元类）
└── Zhi（地支类）
```

**Element（五行基类）：**
- 属性：name, wx（五行）, yinyang
- 方法：getElement(), isYang(), isYin(), getRelationTo(), getWangRelation()
- 静态属性：WX_ORDER, WX_SHENG_ORDER, WX_KE_ORDER

**Gan（天干类）：**
- 继承Element
- 属性：pillarIndex
- 方法：isShiShen(), getTouGanStatus()

**Zhi（地支类）：**
- 继承Element
- 属性：pillarIndex, hiddenGans[], naYin, kongWang, shenSha
- 方法：getHiddenRole(), isAdjacent(), getAdjacentPositions(), isChong(), isHe(), isPo(), isXing()

**Shishen（十神/日元类）：**
- 继承Gan
- 属性：relationToDayMaster, exists[], occurrences[], isWang, isShouZhi, xiYong
- 方法：getName(), isDayMaster(), getTouGanStatus(), calculateWang()

**BaziContext（命盘上下文）：**
- 管理四柱数据、藏干十神、大运流年
- shishenResults[] - 所有十神的出现位置和状态

### 2.2 数据结构

**pillars数组：**
```
[0] 年天干  [1] 年地支
[2] 月天干  [3] 月地支
[4] 日天干  [5] 日地支  ← 日主是 pillars[4]
[6] 时天干  [7] 时地支
```

**shishenResults结构：**
```javascript
{
    ganName: '辛',
    shishen: Shishen实例,
    occurrences: [{ pillar: 0-7, occurs: [天干?,藏干?], role: null/'本气'/'中气'/'余气' }],
    exists: [天干存在?, 藏干存在?]
}
```

## 三、核心概念

### 3.1 十神分类

| 分类 | 十神 | 说明 |
|------|------|------|
| 官星 | 七杀、正官 | 克我者 |
| 印星 | 偏印、正印 | 生我者 |
| 财星 | 偏财、正财 | 我克者 |
| 食伤 | 食神、伤官 | 我生者 |
| 比劫 | 比肩、劫财 | 同类者 |

### 3.2 格局分析

**财官印相生：** 财→官→印（三者相邻，五行相生）

**比肩食伤生财：** 比劫→食伤→财（三者相邻，五行相生）

### 3.3 大运流年相邻关系

大运和流年作为独立柱（位置8-11），与原局所有柱(0-7)都相邻。

## 四、输出规范

### 1. 四柱排盘

```
┌───────────────────────────────────────────────────────────────┐
│ 【四柱排盘】                                                  │
├───────────────────────────────────────────────────────────────┤
│ 年柱: {gan}{zhi} ({十神名称})
│         藏干: {藏干1}({十神1}), {藏干2}({十神2}), {藏干3}({十神3})
│ 月柱: {gan}{zhi} ({十神名称})
│         藏干: ...
│ 日柱: {gan}{zhi} ({十神名称})  ← 日干为日元则十神为空
│         藏干: ...
│ 时柱: {gan}{zhi} ({十神名称})
│         藏干: ...
└───────────────────────────────────────────────────────────────┘
```

### 2. 大运流年

```
┌───────────────────────────────────────────────────────────────┐
│ 【大运流年】                                                  │
├───────────────────────────────────────────────────────────────┤
│ 当前大运: {ganZhi} ({十神})
│ 年龄: {startAge}-{endAge}岁
│ 起始年: {startYear}年
│ 当前流年: {year}年({age}岁) {ganZhi} ({十神})
│ 共 {count} 步大运
└───────────────────────────────────────────────────────────────┘
```

### 3. 基本信息

```
┌───────────────────────────────────────────────────────────────┐
│ 【基本信息】                                                  │
├───────────────────────────────────────────────────────────────┤
│ 日主: {gan} ({wx}, {yy}性)
│ 身强身弱: {level} ({percentage}%)
└───────────────────────────────────────────────────────────────┘
```

### 4. 十神旺衰受制

```
┌───────────────────────────────────────────────────────────────┐
│ 【十神旺衰受制】                                              │
├───────────────────────────────────────────────────────────────┤
│ {gan}: {shishen} | 旺衰={旺衰} | {受制} | 喜忌={喜忌}
│ ...
└───────────────────────────────────────────────────────────────┘
```

### 5. 性格特征

```
┌───────────────────────────────────────────────────────────────┐
│ 【性格特征】                                                  │
├───────────────────────────────────────────────────────────────┤
│ {十神}（{分类}）: {✓正面/✗负面/○中性}
│   {描述}
│ ...
└───────────────────────────────────────────────────────────────┘
```

### 6. 十神格局

```
┌───────────────────────────────────────────────────────────────┐
│ 【十神格局】                                                  │
├───────────────────────────────────────────────────────────────┤
│ [四柱格局]
│ ★ {格局名1}
│ ★ {格局名2}
│
│ 格局明细:
│ 1. 财官印相生（财→官→印）:
│    ★ 匹配! {s1}→{s2}→{s3}
│    {reason}
│ 或
│    ✗ 未形成 - {reason}
│
│ 2. 比肩食伤生财（比劫→食伤→财）:
│    ★ 匹配! {s1}→{s2}→{s3}
│    {reason}
│ 或
│    ✗ 未形成 - {reason}
│
│ [六柱格局] (四柱+大运+流年)
│ ... (同上格式)
└───────────────────────────────────────────────────────────────┘
```

### 验证要点

1. **四柱排盘**：年/月/时柱十神不应为空，藏干十神应正确显示
2. **十神旺衰受制**：每个十神应出现一次，状态明确
3. **性格特征**：应有正面/负面/中性判定和描述
4. **十神格局**：财官印相生和比肩食伤生财应有匹配或不匹配原因
